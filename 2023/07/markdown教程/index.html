Markdown教程

发表于2023-07-28|更新于2023-07-29|写作

前情提要

因为平时不在家，所以文章的更新可能很不及时，而且周末时间很短，估计做不了多少文章，所以才有了今天的教程，教你们一些常用的markdown语法，可以记录一下你们平时的感想，日常，等等，如果想要制作自己的专属页面也可以通过markdown实现，接下来就讲讲如何写作/添加页面，完成之后通过邮箱等联系方式私聊发给我，周末我会部署到网页上。

markdown基础语法

在开始之前，我们需要创建一个文件，格式为xxx.md，然后编辑文本。

文章头部设置

在写文章之前，我们需要引入头部设置，需要设置的项不多，接下来看示例


---
title: markdown    #文章标题
date:               #日期
swiper: true        #主页轮播:true/false
categories:         #分类
tags: []             #标签
top: true           #置顶:true/false

---

需要注意的是，在填入每一项时，后面的内容和引号之间需要有空格。
在填入以上内容后，就可以开始写文章。

标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

隐藏内容

有时候为了有点新意，我们可能需要隐藏文字的效果，比如隐藏提取码之类的，我们可以这么写。


{% hideInline 文本1,文本2,背景色,颜色 %}

#文本1是你想隐藏的内容
#文本2是你想展示的内容，如: 点击查看
#背景色可选: red/blue/green/pink等，背景色指的是按钮的背景色
#颜色同上，颜色指的是按钮上文字的颜色

示例:

{% hideInline 我是清河,我是谁？,green,red %}

效果:

我是清河

hideinline代码功能有限，仅能用来隐藏文字，但是有时候我们想隐藏的可能还有图片之类的，如果你有这种需求的话，block是不错的选择。

block示例代码:


{% hideBlock 按钮文字,背景色,颜色 %}

想要隐藏的文本

{% endhideBlock %}

#配置和上一个很相似，如果不知道的可以去看看hideinline

示例:

{% hideBlock 我是谁？,green,red %}

我是清河

{% endhideBlock %}

示例:

我是清河

相比前两个，hideToggle更像是折叠框，点击可以显示内容，再次点击内容也会收回，接下来是示例代码


{% hideToggle 文本,背景色,颜色%}

折叠框内文本

{% endhideToggle %}

#这里的文本指的是收缩后显示的内容。

示例:

{% hideToggle 我是谁？,red,blue %}

我是清河

{% endhideToggle %}

示例:

我是谁？

我是清河

{% endtabs %}

图片引入

关于图片引入有两种，一种是段落内引入，一种是单独引入。

段落内引入


{% inlineImg 链接 %}

#对于这段代码，也可以加一些参数，调整大小。

{% inlineImg 链接 150px %}

#这里的150ps是你需要改的值，用来控制图片大小。

示例:

这是一段话

这是一段话

直接引入

在不引入段落中间时我们可以采用更简单的方式


![](链接)

#[]内为图片描述，可有可无，链接为图片

示例:

![](http://img.doutula.com/production/uploads/image/2019/08/15/20190815849485_fKMqYD.gif)

#可选背景色:default/blue/pink/red/purple/orange/green

嵌入视频

嵌入视频不算什么特别麻烦的事，目前有一些网站，如blibli支持复制嵌入代码，复制之后粘贴到你想嵌入视频的位置，然后添加参数


<iframe src="链接" width="100%" height="500"></iframe>

#如果是复制的嵌入代码，则仅添加width和height即可

#如果嵌入的只有视频，这里的链接必须是视频直链，如果不理解的，请移步百度。
